# 订单管理中心
## 1.Instruction
**基于mermaid-core**的订单管理中心,提供订单管理业务，它具有如下**功能**
+ 订单查询，支持多条件查询和分页
+ 生成订单
+ 订单修改


## 依赖
- mermaid-framework-core
```xml
<dependency>
        <groupId>com.mermaid.framework</groupId>
        <artifactId>mermaid-core</artifactId>
        <version>1.0-SNAPSHOT</version>
      </dependency>
```
